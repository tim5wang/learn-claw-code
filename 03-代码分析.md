## 🔍 核心功能模块

### 1. Harness 运行时

**职责：**
- Agent 生命周期管理
- 工具调用编排
- 上下文窗口管理
- Token 计数和压缩

**关键特性：**
```rust
// 自动压缩阈值（最新提交）
// fix: auto compaction threshold default 200k tokens
const AUTO_COMPACTION_THRESHOLD: usize = 200_000;
```

### 2. 工具系统（Tools）

**内置工具类型：**
- 文件操作（读/写/搜索）
- Shell 命令执行
- Web 请求
- 代码分析
- 测试运行

**工具注册机制：**
```python
# tools.py - 工具端口元数据
class ToolPort:
    name: str
    description: str
    parameters: dict
    handler: callable
```

### 3. 命令系统（Commands）

**命令分类：**
| 类别 | 示例 |
|------|------|
| 文件操作 | `/read`, `/write`, `/search` |
| 代码操作 | `/edit`, `/refactor` |
| 执行命令 | `/run`, `/test` |
| 信息查询 | `/explain`, `/document` |

### 4. 任务编排（Task）

**任务状态机：**
```
pending → in_progress → reviewing → completed
                ↓
            blocked
```

**持久执行循环：**
- `$ralph` 模式的核心
- 自动重试和恢复
- 架构师级验证

## 🔬 技术亮点

### 1. 干净室重实现

**方法：**
1. 研究原始架构模式
2. 理解 Harness 设计原理
3. 从零实现等效功能
4. 不复制任何专有源码

**合法性保障：**
- 仅学习架构模式
- 不直接复制代码
- 独立实现所有功能

### 2. 双语言策略

**Python 原型：**
- 快速验证概念
- 易于理解和修改
- 社区贡献友好

**Rust 生产：**
- 高性能运行时
- 内存安全保证
- 生产环境就绪
